---> Hashcat

-> Fichier avec le hash ou hash:salt
Le salt est un élément rajouter à la fin d'une string pour rendre le hash incohérent avec une rainbow table. Sans le salt, impossible de déhash un password.
-> hashcat --force -m 1710 -a 0 hash.txt ../rockyou.txt
